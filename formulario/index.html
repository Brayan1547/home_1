    HTML

    HTML funciona mediante etiquetas. Las etiquetas le dicen al navegador que tipo de contenido es cada cosa. Casi todas tienen una apertura < > y un cierre > .

    • < h1 >titulo < h1 > : Define un titulo importante.
    • < p >parrafo < / p > : Define un bloque de texto.
    • < a >Enlace < / a >: Crea un link a otra pagina.
    • < img > :Inserta una imagen.

    STYLES.CSS

    Es un archivo donde vive toda la "personalidad" visual de tu pagina web. mientras qie HTML pone los textos y las imagenes, el archivo CSS les da forma, color y posicion.

    Cada instrucion en el archivo styles.css se divide en tres partes fundamentales.

    • 1. SELECTOR:El nombre del elemento que quieres cambiar.
    • 2. PROPIEDAD:Que quieres cambiar(ej:background-color, font-family, border)
    • 3. VALOR: El ajuste especifico (ej:blue, arial, 2px soild red)

    BOX_MODEL

    Es el concepto fundamental de CSS. Define que cada elemento en una pagina es una caja rectangular. Se compone de cuatro partes:El contenido(texto/imagen), el padding (espacio intermo), el border(el marco) y el margin(espacio externo entre cajas).

    DISPLAY

    Es la propiedad que controla el compotamiento

    y la posicion de las cajas.Determina si un elemento ocupa todo el ancho de la pantalla (block) o solo el espacio de su contenido (inline). Es la clave para decidir como se alinean los elementos entre si.

    FLEXBOX

    Es un sistema de diseño avanzado paara organizar elementos de forma flexible.Permite alinear, central y distribuir el espacio de los elementos de manera automatica y eficiente, sin importar el tamaño FLEXBOX_1 muestra una estructura profecional con carpetas separadas para CSS,HTML e imagenes.

    SELECTORES

    Es la forma en que "llamamos" a los elemntos de HTML desde CSS para darles estilo. Se aprenden selectores de etiqueta (como P o H1), de clase (usando el punto . ) y el ID (usando el numeral #)
